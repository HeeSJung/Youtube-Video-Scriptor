# Repository Description

## Short Description (for GitHub)
Extract YouTube video transcripts with precise timestamp control. Fast, simple, and built with Python.

## Long Description

A lightweight Python tool for extracting transcripts from YouTube videos with timestamp precision. Whether you need a specific clip or the entire video, this tool delivers clean, formatted transcripts in seconds.

### Key Features

- **Timestamp Precision**: Extract exact time ranges (e.g., 10s to 45s)
- **Full Video Support**: Process entire videos in one command
- **Speaker Detection**: Automatically identifies and labels different speakers
- **Multiple Formats**: Outputs in JSON and Markdown
- **Zero Dependencies on FFmpeg**: No video downloads required
- **Fast Processing**: Typically under 5 seconds per extraction
- **Clean Output**: Simple folder structure with no nested directories

### Use Cases

- **Content Creation**: Extract specific clips for quotes, reactions, or analysis
- **Language Learning**: Get transcripts with timestamps for shadowing practice
- **Research**: Analyze video content and dialogue patterns
- **Documentation**: Create text records of video tutorials or lectures
- **Accessibility**: Generate readable transcripts for deaf/hard-of-hearing viewers

### What Makes It Different

Unlike heavy video processing tools, this focuses purely on transcript extraction:
- No video downloading
- No video editing features
- No complex scene detection algorithms
- Just fast, accurate transcript extraction

### Tech Stack

- **Python 3.11+**: Modern, type-safe Python
- **yt-dlp**: Reliable YouTube metadata extraction
- **youtube-transcript-api**: Caption fetching
- **uv**: Fast dependency management

### Perfect For

- Developers building content tools
- Researchers analyzing video content
- Language learners practicing with real content
- Content creators needing quick transcripts
- Anyone who needs YouTube captions with timestamps
